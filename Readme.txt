# JavaTechnology
> Đồ án công nghệ java
## Cách chạy
1. Cài đặt database bằng cách vào thư mục db:
  - Cài đặt MySQL cho máy nếu chưa cài đặt.
  - Cài đặt biến môi trường cho MySQL nếu chưa cài đặt.
  - Chạy file ResetDatabase.bat nếu bạn chạy MySQL8 
  - ResetDatabaseMySQL5.bat nếu bạn đang chạy MySQL phiên bản 5. 
2. Dùng Eclipse để mở source code của website trong thư mục source/uit-java-technology.
